# Collabify

A collaborative web platform where students connect, share what they know, and learn what they don't — completely free, no barriers.

## Overview

Built for campus communities, Collabify lets students list skills they can teach, discover peers offering help in subjects they need, book one-on-one sessions, and grow their reputation through reviews. No tutoring fees, no middlemen — just students helping students.

## Features

- Browse and search skills by subject, category, or availability
- Book peer-to-peer learning sessions with fellow students
- User profiles with skill listings, session history, and reviews
- Messaging inbox for coordinating sessions
- Reputation system through ratings and endorsements

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- No frameworks, no build tools, no dependencies
- Fully static — runs directly in the browser

## Getting Started

No install or build step needed. Clone the repo and open `index.html` directly in your browser.
```bash
git clone https://github.com/mosesamwoma/Collabify.git
cd collabify
open index.html
```

## Pages

| Page | Description |
|---|---|
| `index.html` | Landing page with hero, featured skills, and testimonials |
| `browse.html` | Browse and filter all listed skills |
| `skill-detail.html` | View a skill listing and book a session |
| `profile.html` | Student profile with skills, reviews, and stats |
| `session-book.html` | Schedule and confirm a session |
| `inbox.html` | Messaging between students |
| `login.html` / `register.html` | Authentication pages |
